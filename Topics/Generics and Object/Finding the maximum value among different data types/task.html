Finding the maximum value among different data types

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Create a Java program that applies Generics to receive a blank separated line of three inputs: a string, an integer, and a double. The program should instantiate a generic class that takes these three different data types, and provides a method to get and print the highest value. To test the program, assume the string is only composed of numbers and consider its numeric value, and the highest value is based on the maximum numeric value of the three inputs.
Sample Input:
2 3 2.2

Sample Output:
3.0


Sample Input:
1 1 1.0

Sample Output:
1.0



Memory limit: 256 MB
Time limit: 5 seconds
